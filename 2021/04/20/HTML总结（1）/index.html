HTML总结（1）

本文将介绍一些关于HTML的基础知识

一、HTML介绍

HTML(HyperText Markup Language)超文本标记语言是用来告诉浏览器如何组织页面的标记语言

1、HTML结构

解析：

  1. <!DOCTYPE>：指示 web 浏览器关于页面使用哪个 HTML 版本进行编写的指令。
  2. <html></html>：html元素是文档的根元素，包裹整个页面。
  3. <head></head>：head元素是一个容器，存放meta，title，link等元素，head中的内容不会显示在页面上，用于辅助显示。
  4. <body></body>：body元素包含所有显示在页面中内容，存放结构标签。
  5. <title></title>：用于设置页面标题。当收藏该网页时，此标题将被用来作为网页描述。head标签中必须包含一个title标签
  6. <meta></meta>：meta元素用来设置元数据（元数据简单来说就是描述数据的数据），包括网页使用字符集等。

  • HTML文档中的注释使用<!--和-->来包裹内容，注释中的内容对于用户是不可见的。注释是非常有必要，可以告诉编程者代码的一些相关信息。

  • HTML中的空白

    无论HTML元素的内容有多少空白，最后都会被解析为一个空格，使得文档有更好的可读性。

2、HTML标签

HTML元素和标签的区别：元素包含开始标签,结束标签和内容。

HTML元素分类：

双标签 单标签（自结束标签）
包含开始标签，结束标签和内容 只包含开始标签
分类
块级元素（block） 独占一行，并且不能嵌套在其他内联元素中
内联元素（inline） 也称行内元素，内联元素不会导致文本换行

属性

其中class为属性，name为属性值，属性之间使用空格间隔，建议使用引号包含属性值，单引号和双引号都可以，不使用引号可能会出现意想不到的结果。

布尔属性（属性值和属性名是相同的，可省略属性值）

基本HTML标签

  1. 标题标签

    • h1~h6，对应一级标题到六级标题
    • 一个页面应该只有一个h1标签
    • 常用h2和h3标签

  1. 段落标签

    • 可以描述文章中的一个段落
    • 是一个块元素，独占一行，并且会有一些css默认样式（因浏览器而异）

  1. 文本格式化标签(建议使用第一个)

    • 加粗（<b>,<strong>）

    • 斜体（<i>,<em>）

    • 下划线（<u>,<ins>）

  2. 换行标签（<br>）

    • 使用该标签会强制换行

  3. 水平线标签（<hr>）

    • 会在页面中显示一条水平线
